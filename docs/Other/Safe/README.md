# 安全

分类：

- 私密性：不被非法获取和利用
- 可靠性：不丢失、不损坏、不被篡改

安全问题：

- 用户身份被盗用
- 用户密码被泄露
- 用户资料被盗取
- 网站数据库泄露

## 攻击类型

- XSS 攻击
- CSRF 攻击
- 点击劫持
- 中间人攻击
- Cookies 问题
- 传输安全
- 密码安全
- 接入层注入问题
- 接入层上传问题
- 其他安全
