# nodejs

## what

- 基于 chrome V8 runtime

- 事件驱动

- 非阻塞的 i/o

- i/o:input output 输入输出流，正常下 i/o 的操作都是阻塞的（ajax 同步）

- 网络请求 数据库处理 文件的读写。。。。都属于 IO 操作，阻塞的

- 优点：高并发特别好

## why

1.防止甩锅

2.能够书写 api

3.了解前后端的交互流程

4.全干
