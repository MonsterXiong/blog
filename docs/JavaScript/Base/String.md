# 字符串

## 字符串的常用方法

### 增

用于字符串拼接：

- `+`
- `${}`
- concat()

concat:用于将一个或多个字符串拼接成一个新字符串。

### 删

- slice()
- substr()
- substring()

### 改

- trim():删除所有空格符
- trimLeft()：删除前空格符
- trimRight()：删除后空格符
- repeat()：字符串复制多少次
- padStart()：填充字符
- padEnd()：填充字符
- toLowerCase()、 toUpperCase()：大小写转换

### 查

用于获取字符串的值:

- 直接通过索引
- chatAt()
- indexOf()
- startWith()
- includes()

## 转换方法

split():把字符串按照指定的分隔符，拆分成数组中的每一项

## 模板匹配方法

- match()
- search()
- replace()
